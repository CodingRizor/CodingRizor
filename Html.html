1. HTML stands for Hyper Text Markup Language.
2. HTML document has two parts-

  1. Head part- It contains information about the page such as title.
                          Enclosed by <head> and </head>
                          Header information is not available in the browser window. 
                          It contains the title tag which shows the title of the webpage.
                          <title>Name</title>

   2. Body part- It is where all available text, headlines, graphics are available.
                          Enclosed by <body> and </body>
                          It shows the content available on the page.

Tags

HTML tags are used to mark-up HTML elements and are surrounded by the two character < and > are called angle brackets.
  1. Container tags/Paired tags
    • Tags having both opening and closing.
    • Html elements are defined by starting tags.
  2. Singular tags/Empty tags
    • Tags having only opening like break tag.
    • Html elements which do not need to be closed are known as void elements.

Attributes

Attributes can provide additional information about HTML document.
Attributes have 2 parts:
1. Name
2. Value
Attribute name="value"

Lists

HTML provides two tags to create lists.
  1. <ol>-Ordered List- For numbers
    • An ordered list is a list of items start with numbers,
    • An ordered list starts with <ol> tag.
    • Each list item starts with <li> tag (list item)
    • It has type attribute.
    • <ol type="A/a/I/i/1>

  2. <ul>-Unordered List- For bullets
    • An unordered list is list of items marked with bullets,squares.
    • An unordered list starts with <ul> tag.
    • Each list item starts with <li> tag.
    • Attribute is type.
    • <ul type="disc/circle/square">
Defintion List<dl>
This list is designed to allow for two levels of list item which are empty tag.
<dt>-Definition term - This tag is use to fit on a single line of your webpage.
<dd>-Definition -This tag will accept a full paragraph continuosly beneath <dt> term.

Links

Html provides ability to web to link from one document to another specific site.
Html uses <a> anchor tag to create link to another document or web page.
There are two types of links-
Absolute Links
An absolute link tells exactly where a browser should go by following all the necessary paths.
They are used when we link from one web site to another.
If the html document which we want to link is somewhere else on the internet then absolute url is used.

Relative Links
They are also known as local links.
They just tell the shortcuts for the file.
Its attributes are-
  1. Href attribute
    2. The href attribute is used to tell the address of the document or page linking to it.
    <a href="url"Text to be displayed </a>.
    By clicking on below link you can download sample html notes.
    Html

  2. Target attribute
    3. It defines where the linked document will opened.
    • target="_blank":Opens in a new blank window,
    • target="_self":Opens in current window,
    • target="_parent":Opens in current window.
      • <a href="url" target="_blank">Name of file</a>

  3. Name arttribute
    4. The name attribute is used to create internal links.
    Internal links transfers control to the same page rather than another page.
    <a href="#top/bottom>
    Such as clicking on this link redirect you to top of this window. Go to top
    Such as clicking on this link redirect you to bottom of this window. Go to bottom

Images

The <img> tag displays a graphic image on your page
There are three universally supported Web image formats-
  1. GIF-Graphics Interchanged Format
  2. JPEG-Joint Photographic Experts Group
  3. Portable Network Graphics
Its attributes are-
  1. Src attribute
    2. This src(source) is use to specify the name and the location of your image file.
    Src attribute is compulsory for <img> tag.
    <img src="imagefolder/filename.extension">
    This is a sample image.



  2. Alt attribute
    3. It specify alternative text to appear while the graphics is loading.
    Its value is the image file url.
    <img src="imagefolder/filename.extension" alt="image name" >
    Image with a name.


  3. Align attribute
    4. It is used to align images relative to text.
    <img src="imagefolder/filename.extension" alt="image name" align="right/left/center/justify">
    Image has aligned right.

  4. Border
    5. It is used to create outline of the image.
    <img src="imagefolder/filename.extension" border=number>
    Image has border 2.


  5. Height and Width
    6. Adjust the height and width of the image.
    Height and Width can be in percentage or pixels.
    <img src="imagefolder/filename.extension" height="------" width="---------">
    Image having width="200" and height="80".

Marquee


This is a marquee This tag is used to move image or text on browser window. Its attribute are-
  1. Behavior-This attribute defines the way of moving.
    behavior="scroll"
    behavior="alternate"
  2. Direction-This attribute defines the direction of marquee content.
    direction="left/right/up/down"
  3. Scroll amount
    It is used to define the speed of moving content of marquee tag.
    Value of this attribute is any integer number.
    As the number value is increased the speed of moving content of marquee is increased. scrollamount="5"

Tables

Collection of rows and columns is known as tables.
It starts with <table> and it is a container tag.
It has the following tags-
Table Tags
Tag Description
<table> Defines a table
<th> Defines a table header
<tr> Defines a table row
<td> Defines a table cell
<caption> Defines a table caption
<colgroup> Defines a group of table columns
<col> Defines the attribute values
<thead> Defines a table header
<tbody> Defines a table body
<tfoot> Defines a table footer

Its attributes are-
  1. Align
    The align attribute is used to determine where the table will appear relative to the browser window.
    <table Align=left/right>

  2. Cellspacing
    The cellspacing attribute tells the browser how much space to include between individual cells.
    It is the thickness of the lines making the table grid.
    Value is a number in pixels.
    < table Cellspacing="5">

  3. Cellpadding
    The cellpadding attribute tells the browser how much space to give data elements away from the walls of cell.
    It shows the content away from the border for easy viewing.
    < table cellpadding="5">

  4. Border
    To display a border with table, it is used.
    < table border="5">

  5. Width and Height
    The width attribute defines the width of a table.
    The height attribute defines the height of a table.
    < table width="10%/100">
    < table height="50%/500">

Caption
The table needs a caption to explain the contents.
It is immediately put after the table tag or between the table.
<caption align="top/bottom">name of caption <caption>

Thead
It defines a set of table header rows.
It will appear only once per table at the beginning just after the <table> tag.
It uses <th> tag which boldfaces the text.

Tfoot
It defines a footer in the table.
It will appear only once per table at the end.

Tbody
It collects one or more rows into a group within table.
There can be more than one <tbody> tag in the table.
